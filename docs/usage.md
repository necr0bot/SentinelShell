# Usage

## Start a session

```bash
gpt-session start
```
Do your usual work, scans, etc.

```bash
gpt-session stop
```

This will finalize the log, redact it, and optionally summarize.

## Replay

```bash
gpt-session replay
```

Profile

```bash
gpt-session profile
```
